Dashboard do Gestor apenas com Atividade em Uso

Anteriormente o sistema listava todas as atividades do plano do gestor, porém quando há muitas atividades a apresentação fica muito estranha e com várias linhas.

Implementado inicialmente na versão 3.27

Telas do sistema

Dashboard do Gestor

Figura 1. Lista de atividades (original)
Figura 2. Somente judô e jiu-jitsu em uso
Num novo gestor ainda sem professores, não é possível aferir as atividades em uso, e o sistema mostrará todas as atividades conforme a Lista de atividades (original) até que novos professores sejam cadastrados.
Figura 3. Botão para listar as outras atividades disponíveis
O texto do botão pode ser traduzido para os idiomas que o sistema suporta.
Figura 4. Listando todas as atividades com destaque para atividades em uso

API

O endpoint /stats recebeu uma nova chave:

  • atividades: lista de strings com o nome das atividades em uso baseado nas atividades dos professores cadastrados
