PIX

Levantamento para implementação do recurso de PIX utilizando a API do GerenciaNet.

Esta primeira proposta contempla apenas os recursos mais básicos.

Casos de uso atendimentos

  • Configuração da Gerencianet por Gestor

    • Certificado PIX gerado pela Gerencianet

    • Possivelmente uma chave PIX do gestor (Criar chave evp)

    • Criar PIX para recebimento na Gerencianet

  • Títulos

    • Geração de títulos (gerará PIX de cobrança)

    • Criação manual de título (gerará PIX de cobrança)

    • Edição de título (alterar o PIX de cobrança)

    • Cancelamento de título (alterar o PIX de cobrança)

    • Apresentar QRCode (Gerar QRCode de um location)

    • Apresentar Código de copy/cola (não encontrei este recurso na gerencianet)

    • Baixar “automática” do título (via notificação de pagamento do PIX)

Figura 1. Fluxo do PIX

Dicas de implementação

  • Indicar o uso do boleto ou PIX na configuração (seria global ou por título?)

  • Novo campo para o certificado (backend e web) no gestor, produção e sandbox

  • Novo campo para chave PIX do gestor…​ Cadastro da chave pix?

  • Todo request leva o certificado

  • Criação de um txid para cada título (cobrança), exigindo um novo campo

  • Criação da notificação de pix pago (baixa automática no sistema)
